## macOS真机联调

* 配置adb

```
$ vim ~/.bash_profile
export PATH=${PATH}:~/Library/Android/sdk/platform-tools
export PATH=${PATH}:~/Library/Android/sdk/tools
$ source ~/.bash_profile
$ adb version
```

* 修改adb_usb.ini文件
```
$ system_profiler SPUSBDataType
```
